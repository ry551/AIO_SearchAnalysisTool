# AIO Search Analysis Tool セットアップガイド

本ツールを動かすために必要な各サービスの API キー取得手順を解説します。

## 1. Google Gemini API キーの取得
AI分析（思考部分）を担当します。
1. [Google AI Studio](https://aistudio.google.com/) にアクセス。
2. 「Get API key」をクリックしてキーを作成。
3. 作成したキーをコピーして控えておきます。

## 2. Serper API キーの取得（Google 検索用）
Google 検索結果のデータを取得するために必要です。
1. [serper.dev](https://serper.dev/) にアクセスしてサインアップ（初回無料枠あり）。
2. ダッシュボードから「API Key」をコピーして控えておきます。

## 3. Notion API と データベースの準備
分析レポートの保存先となります。
### ① インテグレーションの作成
1. [My Integrations](https://www.notion.so/my-integrations) にアクセス。
2. 「新しいインテグレーション」を作成。
3. 「シークレットキー（Notion API Key）」をコピーして控えておきます。

### ② データベースの作成と設定
1. Notion で新しい「データベース（フルページ）」を作成。
2. 以下のプロパティ（列）を追加してください（名前が完全一致している必要があります）：
   - **タイトル**: `名前`（デフォルト）
   - **ダイレクト・アンサー**: `テキスト`
   - **独自要素（E-E-A-T）**: `テキスト`
   - **AIO勝因分析**: `テキスト`
   - **構造化ヒント**: `テキスト`
   - **サイト共通ポイント**: `テキスト`
   - **参考URL (AIO引用元)**: `テキスト`
3. 画面右上の「...」メニュー ＞ 「接続先」から、先ほど作成したインテグレーションを招待して追加します。
4. データベースの URL から「データベース ID」を抽出して控えておきます。
   - `https://www.notion.so/{ここがデータベースID}?v=...`

## 4. アプリへの設定方法
取得したキーをプロンプト送信後、AIが作成した `.env.local` ファイルに貼り付けてください。

```env
GOOGLE_GENERATIVE_AI_API_KEY=あなたのキー
SERPER_API_KEY=あなたのキー
NOTION_API_KEY=あなたのキー
NOTION_DATABASE_ID=あなたのキー
```
